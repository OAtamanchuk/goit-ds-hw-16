Це веб-додаток на Streamlit для класифікації зображень одягу з датасету Fashion MNIST за допомогою моделей CNN та VGG16. 
Додаток дозволяє завантажувати зображення, відображати результати класифікації та ймовірності класів.
Додаток реалізує завантаження зображень (PNG/JPG/JPEG), їх передобробку, класифікацію обраною моделлю, вивід передбаченого класу, впевненості та ймовірностей, а також графіки втрат і точності навчання обох моделей. 
Моделі навчені на Fashion MNIST, для великих файлів використовується Git LFS.

Деплой: https://fashionmnistcnnvgg16.streamlit.app
